I began with initializing the project with Vite, creating pure typescript, html/css project. I chose vite due to its fast development workflow (instant update). I chose Babylon.js as the game engine because of its rich library for 3D game development and its relatively small overhead.
I then used claude (desktop version) to go over the task pdf file and create detailed prompt to use in cc (claude code) inside my vs code, in order to achieve a simple initial version. I went on to refine the block design and animations, using blender to create simple beveled cube, and Gemini (Nano Banana) to generate the wood and arrow texture.
